# From Task-Guided Conversational Graphs to Goal-Oriented Dialogue Runtimes
**Status:** Preprint (arXiv)

**Author**

Mariano Garralda-Barrio

## Contribution

This paper introduces **Goal-Oriented Dialogue Runtime (GODR)**, a framework-neutral runtime design pattern for managing long-lived conversational objectives independently of the underlying orchestration technology.

Rather than replacing graph-based workflows, GODR complements them by introducing persistent runtime objects that enable interruption, resumption, invalidation and coordination across multiple concurrent goals.
