## Overview

The **RAG Field Guide** is a practical handbook for engineers, researchers and architects designing Retrieval-Augmented Generation (RAG) systems.

Rather than presenting RAG as a collection of isolated techniques, the guide organizes the design space around the engineering decisions required to build reliable, production-ready systems.

It provides a concise reference covering retrieval strategies, ranking methods, architecture families, evaluation methodologies, production considerations and engineering trade-offs from an evidence-centric perspective.

The guide complements the **Knowledge-Centric AI** research papers and white papers by providing a practical engineering reference for the design, evaluation and operation of modern RAG systems.

---

## Contents

The handbook includes:

- RAG architecture families
- Retrieval methods
- Ranking and reranking
- Retrieval evaluation metrics
- Generation evaluation metrics
- Production metrics
- Architecture selection guidelines
- Engineering trade-offs
- Common failure modes
- Design patterns
- Best practices
- Evidence-Centric RAG
